## Prerequisites

- PHP 8.2 or higher
- Node.js 22 or higher
- Composer
- npm

### Installation

```bash
# Install PHP dependencies
composer install

# Install Node.js dependencies
npm install

# Set up environment file
cp .env.example .env

# Generate application key
php artisan key:generate

# Run migrations
php artisan migrate

# Start development server
composer run dev

# Build for production
npm run build
```
